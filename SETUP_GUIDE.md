## الخطوة 1: إعداد Firebase

مشروعك جاهز: **iqrhq1**

### Firestore Rules
اذهب لـ Firestore → Rules وضع:
```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
```

### Authentication
اذهب لـ Authentication → Sign-in method → فعّل **Email/Password**
